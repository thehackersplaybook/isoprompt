## Quick Start

```python
from isoprompt import optimize_prompt

prompt = "Write a blog post about AI"
optimized = optimize_prompt(prompt)
print(optimized)
```

Or via CLI:

```bash
isoprompt --prompt "Generate a fitness plan for students."
```
